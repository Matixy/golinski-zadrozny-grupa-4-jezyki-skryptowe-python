## 📂 Struktura projektu
```text
JEZYK-SKRYPTOWE/
├── .gitignore               # Plik ignorujący pliki konfiguracyjne i środowiska
├── README.md                # Główne informacje o całym repozytorium
│
├── Lista1_1/                # Katalog z pierwszą listą zadań
├── Lista2/                  # Katalog z drugą listą zadań
│   ├── data/                # Pliki wejściowe (teksty książek)
│   ├── docs/                # Dokumentacja i specyfikacja zadań
│   ├── src/                 # Kod źródłowy aplikacji
│   ├── tests/               # Testy
│   ├── README.md            # Dokumentacja projektu
│   └── requirements.txt     # Lista bibliotek
│
├── Lista3/                  # Analogiczna struktura dla kolejnej listy
└── ...                      # Kolejne foldery (Lista4, Lista5, itd.)
```

##  INSTRUKCJA URUCHOMIENIA PROJEKTU
```
1. WYMAGANIA:
   - Zainstalowany Python (zalecana wersja 3.x).
   - Wszystkie skrypty należy uruchamiać z poziomu głównego katalogu danego projektu.

2. KROKI INSTALACJI:
   - Sklonuj repozytorium na swój komputer:
     git clone <ADRES_REPOZYTORIUM>

   - Przejdź do folderu projektu:
     cd <NAZWA_FOLDERU>

   - Utwórz lokalne środowisko wirtualne:
     python -m venv venv

   - Aktywuj środowisko wirtualne:
     * WINDOWS:       venv\Scripts\activate
     * macOS/Linux:   source venv/bin/activate

   - Zainstaluj wymagane biblioteki (jeśli są obecne):
     pip install -r requirements.txt

3. URUCHOMIENIE PROGRAMU: 
    - Pokazane w plikach README konkretnych projektów
   

```
